## แนวคิดการทำงานใน Google Sheets

ในชีต `เพิ่มข้อมูล` จะมีช่องกรอกข้อมูล และมีแถวตัวอย่าง `A18:R18` ที่เตรียมข้อมูลรูปแบบเดียวกับ `ฐานข้อมูล`

เป้าหมายคือ:

- เมื่อกรอกข้อมูลครบ
- ช่องสถานะ `F14` เป็น `พร้อมบันทึก`
- ระบบจะนำค่าจาก `เพิ่มข้อมูล!A18:R18`
- ไปเพิ่มต่อท้ายชีต `ฐานข้อมูล`
- แล้วล้างช่องกรอกบางส่วนเพื่อพร้อมกรอกรายการใหม่

## Apps Script ที่ใช้

ให้เปิด Google Sheets แล้วไปที่ Extensions > Apps Script จากนั้นวางโค้ดนี้

```javascript
function onEdit(e) {
  const sheet = e.range.getSheet();
  if (sheet.getName() !== "เพิ่มข้อมูล") return;

  const row = e.range.getRow();
  const col = e.range.getColumn();

  const inLeftForm = row >= 4 && row <= 11 && col === 3; // C4:C11
  const inRightForm = row >= 4 && row <= 8 && col === 6; // F4:F8
  if (!inLeftForm && !inRightForm) return;

  const status = sheet.getRange("F14").getDisplayValue();
  if (status !== "พร้อมบันทึก") return;

  appendFormToDatabase_();
}

function appendFormToDatabase_() {
  const lock = LockService.getScriptLock();
  if (!lock.tryLock(5000)) return;

  try {
    const ss = SpreadsheetApp.getActiveSpreadsheet();
    const form = ss.getSheetByName("เพิ่มข้อมูล");
    const db = ss.getSheetByName("ฐานข้อมูล");

    const status = form.getRange("F14").getDisplayValue();
    if (status !== "พร้อมบันทึก") return;

    const values = form.getRange("A18:R18").getValues()[0];
    if (values.every(v => v === "")) return;

    const nextRow = Math.max(db.getLastRow() + 1, 5);
    db.getRange(nextRow, 1, 1, values.length).setValues([values]);

    form.getRange("C4").setValue(new Date());
    form.getRange("C5").setValue(2569);
    form.getRange("C6:C11").clearContent();
    form.getRange("F4").setValue(0);
    form.getRange("F5").setValue(0);
    form.getRange("F6").clearContent();
    form.getRange("F7").setValue("ปกติ");
    form.getRange("F8").clearContent();

    ss.toast("บันทึกเข้าฐานข้อมูลแล้ว", "สำเร็จ", 3);
  } finally {
    lock.releaseLock();
  }
}
```

## วิธีใช้งาน Apps Script

1. วางโค้ดใน Apps Script
2. กด Save หรือ `Ctrl + S`
3. ไม่จำเป็นต้องกด Deploy
4. กลับไป Google Sheets แล้วรีเฟรชหน้า
5. กรอกข้อมูลในชีต `เพิ่มข้อมูล`
6. เมื่อ `F14` เป็น `พร้อมบันทึก` ระบบจะบันทึกเข้า `ฐานข้อมูล` อัตโนมัติ

## จุดที่ต้องระวัง

- ถ้าใน Apps Script ด้านบนขึ้นข้อความ `การเปลี่ยนแปลงที่ไม่ได้บันทึก` ต้องกด Save ก่อน
- ถ้าระบบไม่ทำงาน ให้ดูที่ Apps Script > Executions หรือไอคอนรูปนาฬิกา เพื่อดู error
- แบบ `onEdit(e)` จะบันทึกทันทีเมื่อข้อมูลครบ ถ้ายังต้องการกรอกหมายเหตุก่อน ควรเปลี่ยนเป็นระบบ checkbox เช่น ช่อง `F9` เพื่อกดบันทึกเอง

## สถานะ Google Drive Connector

ระหว่างคุยกัน Google Drive connector ใน Codex ตอบกลับว่า:

```text
401: "Server returned 401: 'Reauthentication required'"
```

จึงยังแก้ Google Sheet ผ่าน Codex โดยตรงไม่ได้ ต้อง reconnect / reauthenticate Google Drive ใน Codex ก่อน ถ้าต้องการให้ AI แก้ชีตให้เองโดยตรง
