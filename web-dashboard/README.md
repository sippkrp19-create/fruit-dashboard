## ฟีเจอร์

- KPI รายรับ รายจ่าย สุทธิ ปริมาณ ราคาเฉลี่ย และเงินค้าง
- เทียบแนวโน้มกับช่วงก่อน
- ข้อความสรุปอัตโนมัติ
- ตัวกรองปี เดือน สินค้า วิธีรับ/จ่ายเงิน และสถานะ
- ตัวกรองสายพันธุ์/เกรด โดยรายการจะสัมพันธ์กับสินค้าที่เลือก และยังจำค่าจากชีต `รายการดรอปดาวน์`
- filter chips ที่กดล้างทีละตัวได้
- กราฟวงกลมเลือกได้ระหว่าง รายรับ/รายจ่าย/สุทธิ
- กราฟวงกลมแยกเป็น 3 วง: ทุเรียน, มังคุด, ลองกอง และแต่ละวงแยกสายพันธุ์/เกรดของสินค้านั้น
- ชนิดสินค้าหลักแสดงเสมอ: ทุเรียน, ลองกอง, มังคุด
- เกรดหลักแสดงเสมอ: เกรด1, เกรด2, เกรด3, เกรด4, เกรดพุ่ม, เกรดปุก, เกรดดำ, เกรดร่วง
- คลิกกราฟเพื่อกรองข้อมูลทั้งหน้า
- ถ้ามีข้อมูลชนิดเดียว donut จะยังวาดเป็นวงเต็มได้
- วิธีรับ/จ่ายเงินรวม `โอน` เป็น `เงินโอน` และไม่แสดง `โอน` เป็นตัวเลือกแยก
- กราฟรายเดือน
- กล่องเงินสด เงินโอน ค้างรับ ค้างจ่าย
- อันดับสินค้า รายจ่าย และเกรด
- ตารางรายการต้องติดตาม
- ตารางฐานข้อมูลล่าสุด
- รองรับมือถือ
